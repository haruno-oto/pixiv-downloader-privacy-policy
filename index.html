プライバシーポリシー (Privacy Policy)

最終更新日: 2025年1月1日

このプライバシーポリシーは、Chrome拡張機能「Pixiv Downloader」（以下、本拡張機能）における利用者情報の取り扱いについて定めたものです。

1. 収集する情報

本拡張機能は、ユーザーの個人情報を外部サーバーへ送信・保存することはありません。
すべてのデータ処理は、ユーザーのブラウザ内（ローカル環境）で完結します。

  • 閲覧データ: Pixiv上の作品情報（画像、タイトル、タグ、ユーザー名など）をダウンロード機能の提供のために一時的に取得しますが、これらは外部サーバーへ送信されません。
  • 設定情報: ユーザーが設定したオプション（保存フォルダ名など）は、Chromeのストレージ機能（API）を使用してブラウザ内に保存されます。

2. 権限の使用目的

本拡張機能は、機能提供のために以下の権限を使用します。

  • activeTab / host_permissions (*://*.pixiv.net/*): Pixivのページ上でダウンローダーを表示し、画像リンクを取得するために使用します。
  • downloads: ユーザーのコンピュータへ画像ファイルを保存するために使用します。
  • storage: ユーザー設定を保存・読み込みするために使用します。

3. データの共有

本拡張機能が取得したデータを、第三者や広告ネットワークと共有・販売することはありません。

4. 免責事項

本拡張機能は、Pixivの公式ツールではありません。本拡張機能の使用によって生じた損害やトラブルについて、開発者は一切の責任を負いません。ユーザー自身の責任においてご利用ください。

5. お問い合わせ

本ポリシーに関するご質問は、Chromeウェブストアのサポートタブよりご連絡ください。

Privacy Policy (English)

Last Updated: January 1, 2025

This Privacy Policy describes how "Pixiv Downloader" (the Extension) handles user information.

1. Information Collection

The Extension does NOT collect or transmit any personal information to external servers.
All data processing occurs locally within your browser.

  • Browsing Data: The Extension temporarily retrieves artwork information (images, titles, tags, usernames, etc.) from Pixiv solely to provide the download functionality. This data is not sent to any external servers.
  • Settings: User preferences are saved locally within your browser using the Chrome Storage API.

2. Permission Usage

  • activeTab / host_permissions (*://*.pixiv.net/*): Used to display the downloader interface on Pixiv pages and retrieve image links.
  • downloads: Used to save image files to your computer.
  • storage: Used to save and load user settings.

3. Data Sharing

We do not share or sell any data retrieved by the Extension to third parties or advertising networks.

4. Disclaimer

This Extension is not an official Pixiv tool. The developer is not responsible for any damages or issues arising from the use of this Extension. Please use it at your own risk.
